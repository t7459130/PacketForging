That's it — exactly what we wanted.
So to summarize the working method for any packet:

Take the Wireshark hexdump
Read bytes 3-4 as little-endian to get the radiotap header length (N)
Strip the first N bytes
Type the remaining bytes into AirpcapReplay as one continuous hex string (no spaces)

That'll give you a correctly-formed 802.11 frame for replay every time.
